Terminology

The text with angle brackets "<>" before and after some text are called tags

The first tag is called the opening tag <> and while the last tags is called the closing tag </> and has a backslash to denote it

The text inside the tags that is being marked is called the content

The combination of opening tag, content, and closing tag is called an element

The text inside of the opening tag is called an attribute, and denotes properties of that element

For example, in <img src="suits.jpg> is the img element with suits.jpg as an attribute

Some elements do not require a closing bracket like img for example
